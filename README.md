# ForgeLog

ForgeLog is a **local-first session and time tracking tool** designed for freelancers, contractors, and solo developers who want accurate billing records without SaaS lock-in.

It creates isolated work sessions, tracks active time, logs commands, and keeps all project artifacts together — entirely on your machine.

No cloud. No accounts. No tracking you don’t control.

---

## What ForgeLog Does

- Start and stop named work sessions
- Track real time spent per project
- Open multiple terminals inside the same session
- Persist sessions even if terminals are closed
- Store logs, notes, and artifacts per session
- One-click GNOME desktop launcher
- Exportable session folders for client delivery
